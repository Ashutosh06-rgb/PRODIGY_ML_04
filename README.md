# 🍱 Food Recognition and Calorie Estimation Model

This project uses deep learning to recognize food items from images and estimate their calorie content, helping users track their dietary intake and make healthier food choices.

## 📁 Dataset

**Food-101**: [Kaggle Link](https://www.kaggle.com/dansbecker/food-101)

- Contains 101 categories of food
- 1,000 images per category
- Already split into training and testing folders

## 💡 Project Goals

- Classify food items using images
- Estimate calorie content based on the predicted food class
